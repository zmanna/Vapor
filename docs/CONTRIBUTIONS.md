## Project Type

Vapor was a collaborative game launcher project. The project should be presented as team work.

## Aryamann Zutshi's Current Contribution Framing

Use this wording unless a more exact commit-level contribution audit is completed:

> I contributed to Vapor as part of a collaborative team, with emphasis on writing, architecture planning, launcher design, and integration direction across connected games and backend services.

## Do Not Claim

- Sole authorship of the full codebase.

- Ownership of all connected game/backend repositories.

- How a launcher coordinates multiple games behind one interface.
- How user, friends, leaderboard, and chat features create cross-game continuity.
- How the team separated launcher pages into modules.
- How integration boundaries were planned between launcher, backend API, and games.
- What should be refactored before production use.
